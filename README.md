# Sales Conversion Rate Dashboard – Power BI

An interactive Power BI dashboard analyzing sales conversion rate across regions, lead sources, product categories, and years.

---

## Dashboard Pages

### Page 1 – Overview
- Total Leads: 1,000
- Total Converted: 486
- Sales Conversion Rate: 48.60%
- Best Selling Month: July
- Leads and conversions by lead source (Referral, Trade Show, Social Media, Website, Email)
- Conversion rate by region (West, Central, South, North, East)

### Page 2 – Deep Dive
- Conversion rate by year (2022 vs 2023) across all regions
- Total converted vs not converted
- Conversion rate by product category (Clothing, Sporting Goods, Home Appliances, Electronics, Books)
- Conversion rate by lead source

---

## Tools Used
- Power BI Desktop
- DAX (Data Analysis Expressions)
- Data Modeling
